# Reinforcement Learning Assignment

This repository is split into two main areas so the assignment work and the module reference material do not get mixed together.

## Layout

- `assignment/code/`: the Python files you will actually run and edit for the assignment.
- `assignment/brief/`: the assignment specification and grading brief.
- `course_materials/lecture_slides/`: lecture PDFs for fast reference while implementing or writing the report.
- `course_materials/notes_and_labs/`: notebooks and lab material that support the module content.
- `course_materials/recommended_reading/`: downloaded supporting reading that is useful background but not part of the runnable assignment code.

## What To Edit

Start in `assignment/code/student_template.py`.

That file is the working assignment script and already imports:

- `assignment/code/environment.py` for the Mastermind environment
- `assignment/code/qtable.py` for the provided Q-table helper

The files under `course_materials/` are reference assets. They are useful for understanding the module and backing up report decisions, but they are not the place to implement the assignment itself.

## Suggested Workflow

1. Read `assignment/brief/Assessment Overview.pdf`.
2. Implement and experiment in `assignment/code/student_template.py`.
3. Use `assignment/code/environment.py` and `assignment/code/qtable.py` as the provided support code.
4. Use `course_materials/` only when you need theory, examples, or lecture context.
